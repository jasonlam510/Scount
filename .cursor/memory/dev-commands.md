# Dev Commands (Scount)

From `package.json`:

- `npm run dev`: Expo dev server
- `npm run start`: Expo dev server (`--go`)
- `npm run ios`: Run iOS
- `npm run android`: Run Android
- `npm run web`: Web (includes PowerSync web assets copy)
- `npm run lint`: Expo lint

Notes:

- Web uses `npm run powersync:copy-assets` before starting.
